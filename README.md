# FitFlow Pro - Gerador de Treinos Profissional

Sistema completo para personal trainers criarem e gerenciarem treinos de academia.

## Funcionalidades

- ✅ Gerador automático de treinos por IA local
- ✅ 150+ exercícios em português
- ✅ Exportar para PDF e Excel
- ✅ Enviar por WhatsApp
- ✅ Login com autenticação
- ✅ Treinos salvos por aluno
- ✅ Interface profissional

## Deploy

Este projeto está configurado para deploy no Render como site estático.
